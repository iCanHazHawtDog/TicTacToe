    System Requirements

    Hardware Requirements

    • x86 Processor or above
    • Minimum hardware required by the Java Virtual Machine
    • Network connection for Network based Multiplayer
    • A Monitor and Mouse

    Software Requirements

    • Any operating system that can run the Java Virtual Machine

    Main Menu

    This is the TicTacToe main application page.

    Access Help

    Help document can be accessed any time.

    High Scores

    This screen gives users the list of TicTacToe high scorers.

    Play

    Play button allows users to initialize the game. On its click, a second screen pops up which leads to initializing a two player game either in the same computer, on the internet or a game played against the computer. To proceed, a player name is required after which either of the options button is selected.

    On this Machine Vs. Human

    Entering the player name and clicking on this button allows you to enter player two's name and start playing the game.

    On this Machine Vs. AI

    Entering the player name and clicking on this button allows you to start playing a game against the computer. In playing against the computer, the difficulty level can be set to either easy, medium or hard. Also in the game, alternating player turns is specified with a counter of wins for both human player and the computer as well as the total number of times the game is played since the game session started.

    Network Based

    Once the player name is specified and this option button is selected, a pop-up window is displayed.

    The message in the window "Waiting for Connection..." means that it is currently in a wait state until another person player on the Internet is interested in a networked game.

    If a networked player is found, the first-In person hosts the game. Once the game session starts, the number of game rounds won is also counted for each player.

    At the end of the game, different messages are displayed to each player and the player hosting the game is expected to continue.

    Pending the time the host player decides to continue the game, the message "Waiting for the first player to continue" is displayed to the second player.

    Play

    After the play board opens, the left top corner has information on the player names and their symbols (X or O), left bottom corner has information on player turn, and the right screen has the board. The players can play the game by clicking on the board cells during their turns only.

    Note: For more detailed help needed in learning how to play the game, click on the Wikipedia link, or watch the YouTube video.

    After each game round, a pop up screen shows up with the game details.

    Reset Button

    Reset button allows a player to reset the game board. All board cells are emptied.

    Exit

    When the exit button is clicked, the game session(s) is/are lost and the game is closed.
